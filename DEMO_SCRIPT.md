# Demo Script

## 20-second version

This is LaunchLens, a real-time Solana launch radar built on Birdeye. It pulls fresh listings, checks trending overlap, enriches tokens with overview data, and ranks them by quality, momentum, and structure. Instead of browsing raw feeds, you get a spotlight token, a ranked leaderboard, and a decision panel in one place.

## 45-second version

LaunchLens is built for one thing: helping traders find better new-token opportunities faster. We use Birdeye new listings to detect fresh launches, Birdeye trending to catch attention shifts, and token overview data to enrich each candidate with liquidity, participation, and momentum. Then we score every token by quality, momentum, and structure, rank the board, and explain why each token is surfacing through catalysts and risk notes. The result is a product that feels more like a decision terminal than an API wrapper.

## Recording flow

1. Start on the hero and radar spotlight
2. Point at the leaderboard and flight score
3. Click one token to open the detail panel
4. Show filters and sort controls
5. Hit `Refresh Radar`
6. End on the methodology section
