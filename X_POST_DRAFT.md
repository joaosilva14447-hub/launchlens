# X Post Draft

Built **LaunchLens** for the @birdeye_data Sprint 1 competition.

It is a real-time Solana launch radar that turns noisy token discovery into a ranked watchlist.

What it does:

- scans fresh listings
- checks trending overlap
- enriches each token with live overview data
- scores quality, momentum, and structure
- surfaces catalysts and risk notes in one decision-ready board

Powered by Birdeye:

- `/defi/v2/tokens/new_listing`
- `/defi/token_trending`
- `/defi/token_overview`

The goal was simple: make new-token discovery feel less like raw feed browsing and more like a usable trading terminal.
